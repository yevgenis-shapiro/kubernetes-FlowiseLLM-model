## Generative AI | Agents and LLM workflows 
Flowise is an open-source, no-code / low-code tool for visually building LLM-based applications ‚Äî like chatbots, RAG (Retrieval-Augmented Generation) systems, agents, and pipelines ‚Äî using a drag-and-drop interface similar to Node-RED or n8n.
